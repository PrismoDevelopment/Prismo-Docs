# Command: Greroll

:::tip Description
`Rerolls a giveaway.`
:::

## Usage

```
greroll <giveaway message id>
```

## Category

_**Giveaways**_

:::info Aliases

- Alias 2: reroll
:::

:::caution Cooldown
