# Command: Snowflake

:::tip Description
`Shows the time difference between between replied message and message ID.`
:::

## Usage

```
snowflake <id>
```

## Category

_**Giveaways**_

:::info Aliases
- Alias 1: timediff
:::

:::caution Cooldown
This command has a cooldown of 5 seconds.
:::
