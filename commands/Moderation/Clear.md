# Command: Clear

:::tip Description
`Clears number of messages in perticular channel`
:::

## Usage

```
clear (type) <amount>
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: purge
:::

:::caution Cooldown
This command has a cooldown of 5 seconds.
:::

## Permissions

+++ User Permissions
- ManageMessages
+++ Bot Permissions
- ViewChannel
- SendMessages
- ManageMessages
- EmbedLinks
+++
