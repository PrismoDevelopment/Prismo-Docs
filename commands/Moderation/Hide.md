# Command: Hide

:::tip Description
`Hides a channel`
:::

## Usage

```
hide <channel>
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: vanish
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::
