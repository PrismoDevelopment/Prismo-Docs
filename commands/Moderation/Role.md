# Command: Role

:::tip Description
`will give roles to server members/bots`
:::

## Usage

```
role <add/all/human/bot> <user/role>
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: addrole
:::

:::caution Cooldown
This command has a cooldown of 5 seconds.
:::

## Permissions

+++ User Permissions
- ManageGuild
- ManageRoles
+++ Bot Permissions
- ViewChannel
- SendMessages
- EmbedLinks
+++
