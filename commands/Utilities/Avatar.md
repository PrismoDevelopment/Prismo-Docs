# Command: Avatar

:::tip Description
`Shows the avatar of the mentioned user.`
:::

## Usage

```
avatar <user>
```

## Category

_**Utilities**_

:::info Aliases
- Alias 1: av
:::

:::caution Cooldown
This command has a cooldown of 5 seconds.
:::
