# Command: Blacklist

:::tip Description
`Will avoid bot from that channel to run any command`
:::

## Usage

```
blacklist <channel/command>
```

## Category

_**Utilities**_

:::info Aliases
- Alias 1: bl
:::

:::caution Cooldown
This command has a cooldown of 5 seconds.
:::

## Permissions

+++ User Permissions
- ManageChannels
+++ Bot Permissions
- ViewChannel
- SendMessages
- EmbedLinks
+++
