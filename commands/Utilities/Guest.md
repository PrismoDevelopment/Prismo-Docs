# Command: Guest

:::tip Description
`Gives Guest role to the user.`
:::

## Usage

```
guest <user>
```

## Category

_**Utilities**_

:::info Aliases
- Alias 1: visitor
- Alias 2: newcomer
- Alias 3: invitee
:::

:::caution Cooldown
This command has a cooldown of 5 seconds.
:::

## Permissions

+++ User Permissions
- SendMessages
- ManageRoles
+++ Bot Permissions
- SendMessages
- ManageRoles
+++
