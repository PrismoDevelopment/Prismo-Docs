# Command: Inviteinfo

:::tip Description
`Get information about an invite`
:::

## Usage

```
inviteinfo <invite>
```

## Category

_**Utilities**_

:::info Aliases

- Alias 2: fetchinvite
- Alias 3: fetchserver
:::
