# Command: Managerrole

:::tip Description
`Used to give Setuproles (bypass)`
:::

## Usage

```
managerrole <role>
```

## Category

_**Utilities**_

:::info Aliases
- Alias 1: setmanagerrole
- Alias 2: manager
:::

:::caution Cooldown
This command has a cooldown of 5 seconds.
:::
