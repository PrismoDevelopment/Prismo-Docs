# Command: Starboard

:::tip Description
`enabling you to highlight and amplify a specific sentence within a designated channel.`
:::

## Usage

```
starboard <enable/disable> <channel> <emoji> <count>
```

## Category

_**Utilities**_

:::info Aliases
- Alias 1: starboard
:::

:::caution Cooldown
This command has a cooldown of 5 seconds.
:::
